# Patch 14 — Correção do arraste de corpos

## Corrigido
- O corpo não continua mais seguindo cliques depois que o botão do mouse é solto.
- Arrastar corpo agora só acontece enquanto o botão esquerdo está pressionado.
- Para arrastar, o corpo precisa estar selecionado antes.
- Clicar e segurar em uma área vazia não move nenhum astro.
- O modo de setas foi separado do arraste com mouse.

## Comportamento correto
1. Clique uma vez em um astro para selecioná-lo.
2. Clique e segure no mesmo astro.
3. Arraste enquanto mantém o botão pressionado.
4. Soltou o botão, o corpo para de seguir o mouse.

## Observação
O botão `Setas` no painel continua servindo para mover o corpo selecionado com as setas do teclado, sem teletransporte por clique.
